# Kero-CV

An intelligent, agentic workflow for parsing candidate CVs, matching them against Job Descriptions (JDs), visualizing gaps, performing simulated ATS checking, and automatically compiling tailored LaTeX CVs and cover letters into PDFs.

---

## 🚀 Features

- **CV Parser**: Extracts raw CV details and compiles them into clean Markdown.
- **JD Parser**: Extracts key roles, skills, and qualifications required by the job.
- **Match Maker**: Analyzes compatibility between the candidate and the JD, generating similarity scores and gap analysis.
- **Visual Analytics**: Plots required vs. possessed skills comparison charts.
- **CV & Cover Letter Writer**: Dynamically drafts polished LaTeX documents.
- **ATS Verifier**: Simulates ATS parsers to verify keyword coverage.
- **Automated Compiler**: Renders LaTeX documents directly to PDF.
